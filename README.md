# Normal Casio Calculator

A responsive normal calculator built using **HTML**, **Tailwind CSS**, and **JavaScript**. The calculator supports basic arithmetic operations along with several scientific functions and a clean interactive user interface.

## Features

### Basic Operations

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)

### Scientific Functions

* Square Root (√x)
* Inverse (x⁻¹)
* Power (xʸ)
* Percentage (%)

### Additional Features

* Repeated "=" operations
* DEL button to remove the last digit
* AC button to clear the current calculation
* ON button to reset the calculator
* Decimal number support
* Operator highlighting
* 12-digit input limit
* Responsive design for mobile and desktop devices

## Technologies Used

* HTML5
* Tailwind CSS
* JavaScript (Vanilla JS)

## Future Improvements
